Practice C++ programming with exercises tailored to your skill level.

## Practice Categories

| Level | Focus | Time |
|-------|-------|------|
| **Easy** | Fundamentals, basic syntax | 30-60 min |
| **Medium** | OOP, STL, algorithms | 1-2 hours |
| **Hard** | Advanced patterns, optimization | 2-4 hours |
| **Interview** | Common interview questions | Varies |

## Usage

```
/cpp-practice [difficulty] [topic]
```

## Examples

```bash
/cpp-practice                    # Random challenge
/cpp-practice easy               # Easy level challenge
/cpp-practice medium stl         # Medium STL exercise
/cpp-practice hard algorithms    # Hard algorithm problem
/cpp-practice interview          # Interview preparation
```

## What Each Exercise Includes

1. **Problem Statement**: Clear description of the task
2. **Input/Output Examples**: Sample test cases
3. **Constraints**: Performance and memory requirements
4. **Hints**: Optional guidance (no spoilers)
5. **Solution Walk-through**: Step-by-step explanation
6. **Complexity Analysis**: Time/space analysis

## Topic Areas

- **Fundamentals**: Variables, loops, functions
- **OOP**: Classes, inheritance, polymorphism
- **STL**: Containers, algorithms, iterators
- **Memory**: Pointers, smart pointers, RAII
- **Algorithms**: Sorting, searching, graphs
- **Modern C++**: C++17/20/23 features
