# C++ Templates Skill

## Overview
Master C++ template programming from basic function templates to advanced metaprogramming techniques and compile-time computation.

## Topics Covered

### Template Basics
- Function templates
- Class templates
- Template argument deduction
- Explicit specialization
- Partial specialization

### Advanced Templates
- Variadic templates
- Template template parameters
- SFINAE and enable_if
- Concepts (C++20)
- Requires expressions

### Metaprogramming
- Type traits
- Compile-time computation
- constexpr functions
- consteval and constinit (C++20)
- Template recursion

### Modern Features
- Fold expressions (C++17)
- Class template argument deduction (CTAD)
- Lambda templates (C++20)
- Abbreviated function templates
- Template parameter packs

### Best Practices
- Template error messages
- Compilation time optimization
- Header organization
- Template debugging techniques

## Prerequisites
- C++ fundamentals
- OOP concepts
- Basic STL usage

## Learning Outcomes
- Write generic, reusable code
- Implement compile-time logic
- Use concepts for constraints
- Create template libraries
